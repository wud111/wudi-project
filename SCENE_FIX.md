# 场景文件修复说明

## 问题
`nian.tscn`场景文件无法打开，可能是文件格式损坏。

## 原因
在多次编辑过程中，场景文件可能：
1. 格式错误（括号不匹配、缩进问题）
2. 节点引用错误
3. 属性设置错误

## 修复方案
创建了一个新的简化版场景文件，包含所有必要元素：

### ✅ 保留的关键元素：
1. **蛇头 (KunzhaHead)** - 带碰撞检测
2. **年兽 (Boss)** - 带攻击系统
3. **UI界面** - 分数显示、游戏结束面板
4. **碰撞系统** - 正确的碰撞层设置
5. **信号连接** - 碰撞信号正确连接

### ✅ 修复的碰撞设置：
```
蛇头 Area2D:
  collision_layer = 2
  collision_mask = 4

年兽和攻击:
  collision_layer = 4  
  collision_mask = 2
```

### ✅ 修复的碰撞形状：
- 蛇头碰撞形状：150×150，位置(0,0)
- 年兽碰撞形状：736×408
- 攻击碰撞形状：适当大小
